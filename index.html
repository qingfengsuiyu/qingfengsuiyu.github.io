DHCP服务器搭建

service network restart #服务重启
mkdir /mnt/cdrom
mount /dev/cdrom /mnt/cdrom

then install dhepd serivice

cd /mnt/cdrom/Packages/
rpm -ivh install dhcp-4.1.1-31.P1 #TABLE补全

编辑DHCPD的配置文件

vi /etc/dhcp/dhcpd.conf

#add below content
ddns-update-style interim;
ignore client-updates;
default-lease-time 21600;
max-lease-time 43200;
subnet 192.168.132.0 netmast 255.255.255.0{
range 192.168.132.100 192.168.132.110;
option subnet-mask 255.255.255.0;
option domain-name “xiaoyi.com”;
option routers 192.168.132.254;
}
​

service dhcpd start
关闭主机，克隆主机，并且启动主机后，启动dhcpd服务，然后编辑克隆的主机，如下命令
vi /etc/sysconfig/network-scripts/ifcfg-eth1 #网卡1

DEVICE=”eht1”
BOOTPROTO=”dhcp”
ONBOOT=”yes”

退出后，执行 ifup eth1 启动网卡1，发现网卡地址已经分配，如果必要启动dhcpd服务

service dhcpd start
